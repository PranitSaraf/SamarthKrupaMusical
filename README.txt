6. Refresh the website

Why password is required now:
- This version does NOT store admin login in localStorage or session storage.
- Every page refresh opens the login screen again.

Important limitation:
- Because this is a static website, the browser cannot directly overwrite script.js on GitHub.
- Downloading a new script.js is the correct static-site workflow.
